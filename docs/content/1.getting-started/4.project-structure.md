## Project Layout

The Movie Recommender project is organized into two main directories:

```bash
movie-recommender/
|-- frontend/            # Nuxt 4 frontend application
|-- docs/                # Documentation site
|-- README.md
`-- LICENSE
```

## Frontend Directory

The frontend is a Nuxt 4 application with Tailwind CSS:

```bash
frontend/
|-- app/
|   |-- app.vue
|   |-- pages/
|   |-- components/
|   |-- composables/
|   |-- layouts/
|   |-- plugins/
|   `-- types/
|-- server/
|   `-- api/
|       `-- tmdb/        # TMDB proxy route
|-- public/
|-- nuxt.config.ts
|-- tailwind.config.js
|-- tsconfig.json
`-- package.json
```

### Key Frontend Areas

- **Pages** - Home, search, watched, profile, login, and reset-password routes
- **Components** - Shared UI such as `MovieDetails.vue`
- **Composables** - Auth, Supabase, and TMDB data helpers
- **Server API** - The current TMDB proxy implementation

## Backend Server

The backend is built into the Nuxt application using Nuxt server routes.

**Current responsibilities:**

- **TMDB Integration** - Proxy requests to The Movie Database API
- **Frontend Support** - Keep the TMDB API key on the server

See [Backend Server](/getting-started/backend) for detailed documentation.

## Documentation Directory

The docs site lives in `docs/` and is built with Docus:

```bash
docs/
|-- content/            # Markdown pages
|-- public/             # Static assets
|-- nuxt.config.ts
`-- package.json
```
