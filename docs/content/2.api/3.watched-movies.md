# Watched Movies API

`/api/watched` stores and reads user watched movies from Supabase table `watched_movies`.

## Auth

- Requires `Authorization: Bearer <supabase_access_token>`
- User is resolved server-side from the token
- Requests without a valid token return `401 Unauthorized`

## Data Source

**Table:** `watched_movies`

- `id`
- `user_id`
- `movies` (`jsonb`, array of `{ tmdbId, title, year }`)
- `updated_at`

## Endpoints

### `GET /api/watched`

Returns current user's watched list.

#### Response

```json
{
  "success": true,
  "movies": [
    { "tmdbId": 550, "title": "Fight Club", "year": 1999 }
  ]
}
```

### `POST /api/watched`

Adds a movie to watched list if not already present.

#### Request body

```json
{
  "movie": {
    "tmdbId": 550,
    "title": "Fight Club",
    "year": 1999
  }
}
```

#### Behavior

- Validates payload (`tmdbId`, `title`, `year`)
- Finds row by `user_id`
- If row exists:
  - appends movie when `tmdbId` is not already in `movies`
  - updates `movies` and `updated_at`
- If row does not exist:
  - inserts new row with current movie in `movies`

#### Response

```json
{
  "success": true,
  "watchedCount": 1
}
```

## Error Codes

- `400` invalid movie payload
- `401` missing or invalid bearer token
- `405` unsupported method
- `500` Supabase query/update/insert failure
