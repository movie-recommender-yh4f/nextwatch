Welcome to **Movie Recommender**, a Nuxt application for browsing movie data from TMDB.

## What is This?

Movie Recommender is currently organized around a lightweight frontend and a small server layer:

- **Backend API** - Proxies TMDB requests from the server
- **Frontend** - Nuxt 4 with Vue and TailwindCSS
- **Authentication** - Supabase-based sign-in and password reset flows
- **Docs** - Docus-based developer documentation

## Documentation Structure

This documentation focuses on the current technical implementation:

- **Getting Started** - Setup, environment variables, and project layout
- **Backend API** - Current server routes and TMDB proxy behavior
- **Frontend** - Vue components, pages, routing, and composables
