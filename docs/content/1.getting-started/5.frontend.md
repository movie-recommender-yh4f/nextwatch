## Frontend Setup

Located in `frontend/` directory.

### Install

```bash
cd frontend
npm install
```

### Environment Variables

Create `.env.local`:

```text
NUXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NUXT_PUBLIC_SUPABASE_ANON_KEY=your-key
NUXT_TMDB_API_KEY=your-tmdb-key
```

`NUXT_PUBLIC_SUPABASE_*` powers client-side auth. `NUXT_TMDB_API_KEY` is used by the server-side TMDB proxy.

### Run Dev Server

```bash
npm run dev
```

Available at `http://localhost:3000`

### Build

```bash
npm run build
npm run preview
```
