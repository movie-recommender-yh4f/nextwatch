# TMDB Proxy API

The TMDB Proxy allows frontend to securely request data from The Movie Database API without exposing API key to the client.

## Endpoint

```text
GET /api/tmdb/:path
```

## How It Works

1. Client sends request to `/api/tmdb/...`
2. Server receives the request and extracts the path
3. Server adds `NUXT_TMDB_API_KEY` from environment variables
4. Request is forwarded to TMDB API with all query parameters
5. Response is returned to client

## Example Usage

### Get Movie Details

```js
// Instead of : https://api.themoviedb.org/3/movie/550?api_key=xxx

// Use:
const response = await fetch('/api/tmdb/movie/550')
const movie = await response.json()
```

### Search Movies

```js
const query = 'The Matrix'
const response = await fetch(`/api/tmdb/search/movie?query=${encodeURIComponent(query)}`)
const results = await response.json()
```

### Get TV Show Details

```js
const response = await fetch('/api/tmdb/tv/1396') // Breaking Bad
const show = await response.json()
```

## Request Format

### Path Parameter

- **path** - The TMDB API path (e.g., `movie/550`, `search/movie`, `tv/1399`)

### Query Parameters

Pass any TMDB API parameters as query strings:

- `query` - Search string
- `page` - Results page number
- `sort_by` - Sort order
- `with_genres` - Genre filter
- etc. (all TMDB parameters supported)

### Headers

The following headers are automatically set by the proxy:

```text
Accept: application/json
api_key: [from TMDB_API_KEY env]
language: en-US
```

## Response Format

Returns whatever TMDB API returns in JSON format.

**Example Movie Response:**

```json
{
  "id": 550,
  "title": "Fight Club",
  "overview": "A ticking-time-bomb insomniac...",
  "poster_path": "/path/to/poster.jpg",
  "release_date": "1999-10-15",
  "vote_average": 8.8,
  "genres": [...]
}
```

## Error Handling

If TMDB API returns an error, the proxy forwards the error response:

```json
{
  "error": "Invalid API key",
  "statusCode": 401
}
```

## Environment Setup

Make sure to set the TMDB API key:

```bash
# .env
NUXT_TMDB_API_KEY=your_api_key_here
```

Get your key from [themoviedb.org/settings/api](https://www.themoviedb.org/settings/api)

## Security Notes

- API key is never exposed to the client
- Requests are validated on the server
- All queries are forwarded to TMDB
- Language is always set to English (configurable if needed)
