## Prerequisites

Make sure you have the following installed:

- **Node.js** (v18 or higher)
- **Git**
- **npm**

## Installation Steps

::steps
### Clone the Repository

```bash [Terminal]
git clone https://github.com/movie-recommender-yh4f/movie-recommender.git
cd movie-recommender
```

### Setup Frontend

Navigate to the frontend directory and install dependencies:

```bash [Terminal]
cd frontend
npm install
```

Configure environment variables by creating a `.env.local` file:

```bash [Frontend .env.local]
NUXT_PUBLIC_SUPABASE_URL=your_supabase_url
NUXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
NUXT_TMDB_API_KEY=your_tmdb_api_key
NUXT_GEMINI_API_KEY=your_gemini_api_key
NUXT_LIBSQL_URL=file:../database/local.db
```

To obtain the Gemini API key, visit [https://aistudio.google.com/api-keys](https://aistudio.google.com/api-keys).

### Start Development Server

Start the frontend application:

```bash [Terminal]
npm run dev
```

The application will be available at `http://localhost:3000`

### Optional: Run Documentation Locally

If you want to work on the docs site too:

```bash [Terminal]
cd ../docs
npm install
npm run dev
```
::

## Verify Installation

Once everything is set up, you should be able to:

1. Open `http://localhost:3000` in your browser
2. Browse popular titles on the home screen
3. Open movie details, search titles, and access the auth pages

## Troubleshooting

If you encounter any issues:

- Ensure all dependencies are correctly installed
- Check that environment variables are properly configured
- Verify that Node.js version meets the requirements
- Check the [Troubleshooting](/getting-started/troubleshooting) guide for common issues
